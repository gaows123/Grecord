构造路径规划器

参数
world_model路径规划器用来碰撞检测的世界模型
costmap局部规划器用到的代价地图的引用
footprint_spec代表机器人的多边形(必须是凸的)
inscribed_radius机器人内切圆的半径
circumscribed_radius机器人外接圆的半径
acc_lim_x机器人在x方向上的加速度限制
acc_lim_y机器人在y方向上的加速度限制
acc_lim_theta角速度限制
sim_timeThe number of seconds to "roll-out" each trajectory "roll-out"每个局部路径的秒数
sim_granularity仿真点之间的距离应该足够的小而不会撞到障碍物
vx_samples在x方向采样的局部路径数量？
vtheta_samples在旋转方向采样的局部路径数量
path_distance_bias调节机器人应该离全局路径多近的尺度因子
goal_distance_bias调节机器人应该多'积极'地导航去目标点的尺度因子
occdist_scale调节机器人离障碍物距离尺度因子
heading_lookaheadHow far the robot should look ahead of itself when differentiating between different rotational velocities？
oscillation_reset_distThe distance the robot must travel before it can explore rotational velocities that were unsuccessful in the past
escape_reset_dist退出逃跑模式之前机器人要走过的距离长度
escape_reset_theta退出逃跑模式之前机器人要旋转的角度阈值
holonomic_robot如果为true, 机器人也会被y方向的线速度控制
max_vel_x最大线速度
min_vel_x最小线速度
max_vel_th最大角速度
min_vel_th最小角速度
min_in_place_vel_th最小定点旋转角速度 The absolute value of the minimum in-place rotational velocity the controller will explore
backup_vel后退速度？ The velocity to use while backing up
dwa如果为true, 将用动态窗口法DWA
heading_scoringtrue表示轨迹打分项会包括机器人朝向
heading_scoring_timestep从多远开始为局部轨迹的朝向打分
meter_scoring调节参数和代价地图的分辨率适配
simple_attractortrue允许 to allow simple attraction to a goal point instead of intelligent cost propagation
y_velsy方向线速度的向量
angular_sim_granularity仿真点之间的距离应该足够的小而不会撞到障碍物？
